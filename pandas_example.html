Energy Spectra¶

Let's start by taking a look by plotting some energy spectra for each channel. If we want to plot all channels in one figure, we can just use the default hist function in pandas.

In [3]:
bins = np.arange(250, 2650, 1) #1 keV bins from [250,2650)

#Split the dataframe into groups for each unique channel, and plot an energy spectrum for each
for chan, df_chan in df.groupby("channel"):
    df_chan.ecal.hist(histtype="step", bins=bins, figsize=(8,5), label="Channel {}".format(chan))
plt.yscale("log")
plt.legend()

Seaborn makes it easy to split data across a category and make an individual plot for each channel.

In [4]:
g = sns.FacetGrid(df, col="channel", margin_titles=True, size=5, aspect=1.4)

g.map(plt.hist, "ecal", color="steelblue", bins=bins, lw=1, histtype="step")
g.set(yscale="log", xlabel="Energy [keV]", xlim=(250,2650))
